What is it?

This web app will help you create Invoice, Receipt and more with fast and decent. You will be able to adjust tax rate, language, currency and more.

This is kinda my first web app and I'm looking forward to continue update a better version. I hope this is gonna help you work with document stuff. It is FREE to use for everyone, if you like and want to support me, you can donate by clicking the link at the end of the page.

How to use?

  • Choose the template you want to use
  • Fill up the form you've chosen
  • Setting the form style as your wish
  • [Ctrl + P] to preview (Chrome)
  • When you are done, click [Print]
  • You will see the long URL, save or copy it to use next time
